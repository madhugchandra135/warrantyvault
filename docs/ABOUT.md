**Problem:** People lose receipts required for warranty claims. Paper bills fade or get misplaced.  
**Solution:** A digital vault that stores, scans, and tracks warranty periods.

**Core functions**
- Upload any bill (photo/PDF)
- OCR auto-fills vendor, date, and warranty duration
- Dashboard view for all products + expiry countdown
- Notifications for soon-to-expire warranties

**Future features**
- Email-to-vault (auto-capture invoices from Gmail)
- Reminder push notifications
- API for retailers
- Subscription for cloud sync
